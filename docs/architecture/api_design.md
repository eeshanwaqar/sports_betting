# API Design

## Overview

This document describes the REST API design.

## Endpoints

### Predictions
- POST /api/v1/predict
- POST /api/v1/predict/batch
- GET /api/v1/predict/upcoming

### Teams
- GET /api/v1/teams
- GET /api/v1/teams/{name}

### Matches
- GET /api/v1/matches/history
- GET /api/v1/matches/head-to-head

### Chat (RAG)
- POST /api/v1/chat
